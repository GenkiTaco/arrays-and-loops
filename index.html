Arrays and Loops

Arrays

An array is a data structure used to store multiple values in a single variable. The values within an array are indexed, meaning each element can be accessed by its position (index).

Indexed Arrays

  • Indexed arrays use numeric (integer) positions to identify elements.
  • The first element is often at index 0 in languages like JavaScript, C, C++, Java, and Python.
  • The next element is at index 1, and so on.

Associative Arrays

  • Associative arrays use named keys to identify elements.
  • Keys can be strings or integers.
  • Values can be any data type, including strings, numbers, booleans, objects, and other arrays.

Multidimensional Arrays

  • Multidimensional arrays are arrays within arrays.
  • They are used to store complex data structures, such as tables, matrices, and lists of lists.
  • Each element can be accessed by multiple indices.

Array Functions

  • Arrays have built-in functions for adding, removing, and sorting elements.
  • Common functions include push(), pop(), shift(), unshift(), splice(), and sort().
  • These functions can be used to manipulate arrays in various ways.

Sorting Arrays

  • Arrays can be sorted in ascending or descending order.
  • Sorting can be done based on numeric or string values.
  • Custom sorting functions can be defined for complex sorting requirements.

Loops


For Loop

  • The for loop is used to iterate over a sequence of elements.
  • It consists of an initialization, condition, and increment/decrement expression.
  • The loop continues as long as the condition is true.

While Loop

  • The while loop is used to execute a block of code as long as a condition is true.
  • It consists of a condition that is evaluated before each iteration.
  • The loop continues as long as the condition is true.

Foreach Loop

  • The foreach loop is used to iterate over elements in an array or object.
  • It automatically assigns each element to a variable for processing.
  • It simplifies the process of iterating over arrays and objects.

Loop Control

  • Loop control statements like break and continue can be used to alter the flow of a loop.
  • Break terminates the loop early, while continue skips the current iteration.
  • These statements can be used to handle special cases or conditions within a loop.

Loop Functions

  • Loops can be used to perform repetitive tasks, such as iterating over arrays or objects.
  • Functions can be defined inside loops to encapsulate logic and improve readability.
  • Functions can be called within loops to perform specific actions on each iteration.
